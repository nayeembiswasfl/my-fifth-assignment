1️⃣ What is the difference between var, let, and const?
Answer:
var, let and const all used to make variable in JavaScript. But they not exactly same.
var is old way. It can be re-declare and also you can change the value many time.
let is newer than var. You can change the value but you cannot declare it again in same block.
const is for value that should not change. Once you give value, later you cant change it.

So simple way: var = old, let = changeable variable, const = fixed value.

2️⃣ What is the spread operator (...)?
Answer:
Spread operator is written like three dots .... It is used to spread or copy values from array or object. It help to combine things or copy them easily. Basically it take all items and expand them.

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer:
All three work with array and go through each item. But they do different work.

map() go through array and make a new array after changing each value.
filter() go through array and keep only the items that match condition.
forEach() just go through array and run something on each item but it dont return new array.

4️⃣ What is an arrow function?
Answer:
Arrow function is shorter way to write function in JavaScript. It use => symbol. It make code smaller and more simple to write. Many developers use it because it look cleaner.

5️⃣ What are template literals?
Answer:
Template literals are special way to write string in JavaScript using backticks ` instead of normal quotes. It allow to put variables inside string easily using ${ }. Also it make writing long or multi line text easier.
